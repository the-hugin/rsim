# /new-feature

Реализуй фичу: $ARGUMENTS

## Инструкции (для небольших фич без PRP):

1. Прочитай CLAUDE.md
2. Проверь `examples/` на похожие паттерны
3. **Предложи подход в 3-5 строк** — жди подтверждения перед кодом
4. После подтверждения — запусти **Task tool** (`general-purpose`) с изолированным контекстом:
   - Передай содержимое CLAUDE.md, описание задачи, подход из шага 3
   - Scope Rules и команды валидации передай в промпте
   - Дождись результата и выведи итог в основной чат

## Scope Rules:

- Изменяй только файлы напрямую связанные с задачей
- Не рефакторь то, что не сломано
- Не добавляй зависимости без явного согласования
- Не создавай новые абстракции если можно использовать существующие

## После реализации:

1. Запусти валидацию — команды возьми из секции `ENVIRONMENT` в `CLAUDE.md`.
   Если CLAUDE.md не найден или ENVIRONMENT пуст → используй дефолт по стеку:
   - TypeScript/React: `npm run lint && npx tsc --noEmit && npm run test`
   - Python: `ruff check . && python -m pytest`
   - Electron: `npm run lint && npx tsc --noEmit`
2. Выведи список изменённых файлов
3. Предложи обновить CLAUDE.md Session Log

---
> Для фич > 2 часов используй `/prp-create` вместо этой команды.
